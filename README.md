# Melodia Wellness Hub

Melodia Wellness offers holistic wellness coaching — personalized planning, stress management, mindfulness training, and lifestyle coaching. This repository contains the source code for the Melodia Wellness web application.

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
2. **Install dependencies**:
   ```sh
   npm install
   ```
3. **Start the development server**:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Framework**: Vite + React
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Backend**: Firebase
- **Animations**: Framer Motion

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Individual page components.
- `src/lib`: Configuration and utility functions (e.g., Firebase, common utils).
- `public`: Static assets like favicons and images.

## Deployment

The project is deployed via a standard static site hosting provider. To build for production, run:
```sh
npm run build
```
